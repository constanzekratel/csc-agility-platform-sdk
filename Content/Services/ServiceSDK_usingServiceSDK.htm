Using the Services SDK

DXC Technology provides the Services SDK so that you can build custom adapters for integration with any external service required by an application or the infrastructure on which it is deployed.

Using the Services SDK to develop and publish a service adapter for Agility Platform typically involves the following workflow:

1. Define the adapter components.
2. Register the adapter.
3. Synchronize the adapter.

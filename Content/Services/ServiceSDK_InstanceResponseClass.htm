InstanceResponse Class

The Services SDK InstanceResponse class is wrapped by the promise returned by the InstanceLifecycle methods. InstanceResponse extends ServiceProviderResponse with the following methods:

Method

Type

Description

getInstance()

Instance

Returns the modified instance

The value is null if not set.

setInstance(Instance value)

void

Used by the service adapter to set the modified instance from the request that triggered the response

The Services framework subsequently persists the instance asset. The service adapter can skip setting the instance in the response if it does not modify the instance from the request.

InstanceRequest Class

The Services SDK InstanceRequest class is the basis for the parameter to the IInstanceLifecycle methods. InstanceRequest extends ServiceProviderRequest with the following methods:

Method

Type

Description

getInstance()

Instance

Returns the VM instance

getTemplate()

Template

Returns the template for the instance

getOperatingSystem()

OperatingSystem

Returns the operating system for the VM instance

getServiceInstance()

ServiceInstance

Returns the service instance asset connected to the VM instance

getPeerServiceInstances()

List<ServiceInstance>

Returns a List of service instances that are the destination of connections to the template associated with the instance

getSrcConnections()

List<Connection>

Returns a List of all connections from the service instance to another Agility Platform asset

getDestConnections()

List<Connection>

Returns a List of all connections to the service instance from another Agility Platform asset

getDependencies()

List<Asset>

Returns a List of all Agility Platform assets upon which this service has a dependency

getDependents()

List<Asset>

Returns a List of all Agility Platform assets dependent upon this service

getLaunchItemDeployment()

LaunchItemDeployment

Returns, if applicable, the Launch Item Deployment for this service

getVariables()

List<AssetProperty>

Returns a list of variables inherited from the containment hierarchy

Note: Only one value for any variable name is passed. Standard script variable inheritance is used to determine which value is passed if the same variable exists on multiple containers in the parent chain of the service instance. Values for encrypted variables will be decrypted in this list.

Note:  Access to variables in the scope of a ServiceInstance (variables on Projects, Environments, and Topologies that are direct ancestors of the ServiceInstance) is provided for ServiceInstanceOperations, ServiceInstanceLifecycleOperations, InstanceOperations, and ConnectionOperations methods. The variables can be found in the variables list on the request object (request.getVariables). Encrypted variables in the list are passed as decrypted. There is no support for accessing variables outside of these methods or outside the scope of the ServiceInstance. If you have a unique use case that requires access to these out-of -scope variables, use the Agility Platform API to access them. For example: List<AssetProperty> vars = request.getVariables();
